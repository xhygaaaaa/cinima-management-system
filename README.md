影院管理系统
 
项目概述
 
这是一个基于Java和MySQL的影院管理系统，主要用于影院日常运营管理，包括用户管理、电影管理、影厅管理、排片管理、票务管理、会员管理、卖品管理和数据统计等功能。
 
功能模块
 
1. 用户管理：实现用户登录、注册和权限管理。
2. 电影管理：维护电影信息，如添加、修改、删除电影。
3. 影厅管理：管理影厅信息，包括影厅编号、座位数等。
4. 排片管理：安排电影放映计划，设置放映时间、影厅等。
5. 票务管理：处理售票、选座等业务。
6. 会员管理：管理会员信息，包括积分、优惠等。
7. 卖品管理：管理卖品信息，如可乐、爆米花等。
8. 数据统计：统计票房、会员消费等数据。
 
技术栈
 
- 后端：Java
- 数据库：MySQL
